# 功能完成度矩陣（作品展示版）

> 範圍：不含真實金流、Email、SMS、QRCode 等外部服務，僅以站內 mock / in-app 方式展示。

## 核心使用者流程

| 模組 | 功能 | 狀態 | 備註 |
|---|---|---|---|
| 首頁與輪播 | 10 部電影輪播、場次入口、會員/員工浮動選單 | 已完成 | 前端資源改為本機 webjars 載入 |
| 會員登入 | 登入/登出、錯誤提示、帳號鎖定 | 已完成 | 支援失敗次數鎖定 |
| 員工登入 | 登入/登出、角色導向頁面 | 已完成 | IT / 主管 / 管理員分層 |
| 雙身份並存 | 同瀏覽器同時會員+員工登入 | 已完成 | 兩套獨立 SecurityContext |
| 訂票流程 | 選位、規則校驗、建立訂單 | 已完成 | 單筆最多 4 張等限制 |
| 付款流程（Mock） | 成功/失敗/逾時 | 已完成 | 不接外部金流 |
| 訂單中心 | 訂單列表、明細、取消 | 已完成 | 30 分鐘取消限制 |
| 會員通知 | 站內通知、已讀、最近回饋 | 已完成 | 保留 30 天自動清理 |
| 會員點數 | 累積、兌換、紀錄 | 已完成 | 依付款金額累積 |
| 員工檢查表 | 每日檢查表、批次操作 | 已完成 | 後台可操作 |
| 維修申請 | 提交、狀態流轉、關閉 | 已完成 | OPEN -> IN_PROGRESS -> RESOLVED -> CLOSED |
| 場次管理 | CRUD / 停用 | 已完成 | 管理員權限 |
| 電影管理 | CRUD / 上下架（CMS） | 已完成 | 管理員權限 |
| 多影城欄位 | 場次支援地點編碼/名稱 | 已完成 | migration V25 |
