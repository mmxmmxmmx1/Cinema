# 3 分鐘 Demo 劇本

## 準備

1. 啟動系統：`mvn spring-boot:run`
2. 開首頁：`http://localhost:8080`
3. 用測試會員登入（例如 `test123 / test123`）

## 流程 A：訂票到付款

1. 選電影與場次，最多選 4 張座位。
2. 進入結帳頁，選「付款成功（mock）」。
3. 顯示成功訊息後，前往「我的訂單」。
4. 在「進行中訂單」看到新訂單；會員專區也同步顯示「即將欣賞」。

## 流程 B：取消與釋位

1. 進入該筆訂單詳情，點「取消訂單」。
2. 返回訂單頁確認狀態為「已取消」。
3. 回到選位頁，原座位可再次選取（確認釋位成功）。

## 流程 C：通知與點數

1. 進入「站內通知」頁面，看到建立訂單/付款/取消等通知。
2. 測試單筆已讀與「全部標記已讀」。
3. 到「點數紀錄」頁確認付款累點、取消回滾。

## 流程 D：員工與管理功能

1. 員工登入 `/employee/login`，查看今日待辦與場次概況。
2. 主管頁 `/employee/manager/dashboard` 查看營運 KPI。
3. 管理頁 `/employee/admin/dashboard` 看角色與營運指標。
4. 維護工具 `/employee/admin/tools` 觸發清理操作。
